#  第3章 Andriod 编译系统
## 3.1 Makefile入门
```
makefile


target: prerequisites
    commands

target: 需要生成的目标文件
prerequisites：生成目标文件需要的所有依赖
commands
    note:prerequisites任何一个文件比target 新都会触发commands 这些命令的执行



makefile 例子
main.out:main.o opt.o
	g++  -o main.out main.o opt.o


main.o:main.cpp opt.hpp
	g++ -c main.cpp

opt.o:opt.cpp opt.hpp
	g++ -c opt.cpp
    
run:
	./main.out



makeflle 变量
note:
    传统的变量全部都是大写,建议使用开头字母大写因为这样可以避免和系统变量冲突


```
