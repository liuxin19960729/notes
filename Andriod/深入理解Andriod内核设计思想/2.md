## 2.8 Andriod 反编译
```
apk -> zip
apk 解压 里面有classes.dex(是Dalvik JVM虚拟机支持的可执行文件)

1.
apk 解压就可以将 *.dex 或 *.odex 文件弄出来
2.
system.img (framework) 代码怎样讲文件还原出来
    recovery模式 


工具
baksmali 将 odex 转成 dex

smaili 将 dex 转成java文件
dex2jar 将 dex转成jar

jd-gui 阅读jar java 等代码
https://java-decompiler.github.io/
```
