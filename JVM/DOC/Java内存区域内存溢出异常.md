### 2.2.4 Java 堆
```
《Java 虚拟机规范》 对Java堆描述 所有的对象实例和数组都应该在堆上分配内存

垃圾收集器管理Java 对内存区域
线程共享
《Java 虚拟机规范》 规定 JAVA 对的内存在物理内存上可以存在不连续的内存上,但是在逻辑上因该将它视为连续内存

Java 堆大小既可以实现可扩展的和固定大小的
note:对不能扩展并且无法在分配新内存会抛出OutOfMemoryError 异常
```
